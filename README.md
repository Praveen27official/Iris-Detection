🌸 Iris Flower Detection using Machine Learning

A Machine Learning project that classifies Iris flowers into different species based on their petal and sepal measurements using Python and popular data science libraries.

## 🚀 Overview

The Iris Flower Detection project is one of the most popular beginner-friendly machine learning projects.  
This project uses the famous Iris dataset to train a classification model capable of predicting the species of an iris flower.

The model classifies flowers into:
- Iris-setosa
- Iris-versicolor
- Iris-virginica

---

## 📊 Dataset

The dataset contains:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width
- Species

Dataset Source:
- Scikit-learn Iris Dataset
